# Knowledge Base para Restaurantes

<div class="hero-wrapper">

Bienvenido a tu base de conocimiento colaborativa para la industria de la restauración. Aquí encontrarás manuales operativos, guías de sistemas, habilidades prácticas y un asistente IA que responde tus dudas al instante.

</div>

## ¿Qué encontrarás aquí?

<div class="features-grid">

<div class="feature-card">

### Manuales operativos

Protocolos detallados para la apertura de un restaurante, operaciones diarias y gestión financiera. Documentación probada en entornos reales de alta exigencia.

</div>

<div class="feature-card">

### Sistemas de gestión

Metodologías para el control de costes, cálculo de food cost, gestión de inventario y optimización de mermas. Todo basado en datos reales.

</div>

<div class="feature-card">

### Skills para el equipo

Técnicas de atención al cliente, resolución de conflictos y comunicación interna. El factor humano marca la diferencia.

</div>

<div class="feature-card">

### Asistente IA

Pregunta en lenguaje natural sobre cualquier manual. El asistente busca en toda la base de conocimiento y te devuelve respuestas con cita directa a la fuente.

</div>

</div>

## ¿Cómo usar esta base de conocimiento?

Navega por las secciones del menú superior o utiliza el **buscador** para encontrar el contenido que necesites. Si tienes una duda concreta, prueba el asistente IA: te ahorrará tiempo buscando entre documentos.

> **Sugerencia:** Empieza por [Operaciones diarias](manuales/operaciones-diarias.md) si ya tienes el restaurante abierto, o por [Apertura](manuales/apertura.md) si estás en fase de lanzamiento.
